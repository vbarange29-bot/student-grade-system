# Student Grade Management System

A simple C++ mini project that demonstrates Object-Oriented Programming (OOP), STL Vector, and File Handling.

## Features
- Add student records
- Store student names and marks
- Display student records
- Save records in students.txt

## Technologies Used
- C++
- OOP (Classes & Objects)
- STL Vector
- File Handling

## Files
- student_grade_system.cpp
- students.txt
- README.md

## Conclusion
This project demonstrates the use of Object-Oriented Programming (OOP), STL Vector, and File Handling to manage student records in C++.
